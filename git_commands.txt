GIT COMMANDS

I. First step

1) git init  (for initialization of the repository)
2) git add .   (. means any file extension)
	git add *.java (all files only with java extensions)
3) git commit
	git commit -m "pass some message"
4) git remote add origin <remote url>
Eg: git remote add origin https://github.com/SharanyaKishtapuram-star/VISEM_JFS.git
5) git push origin master(old version)
	git push origin main(new version)
6) you can heck it in remote repository
